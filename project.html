Tic-Tac-Toe

The project involved the creation of a web-based tic-tac-toe application that facilitates user-to-user gameplay. JavaScript was employed to develop functions responsible for managing user inputs, monitoring game progression, and validating winning scenarios. Additionally, a reset button was integrated to enable the initiation of new game sessions. The primary objective of the game is for a player to achieve a sequence of three markers in a row, either horizontally, vertically, or diagonally, in order to secure victory.

Weather App

This project entailed the development of a web page that retrieves weather data from a weather API (Application Programming Interface) based on the user's current location or a user-inputted location. The page incorporates buttons, an input field, and an aesthetically pleasing design using Cascading Style Sheets. JavaScript was utilized to obtain weather information, including atmospheric conditions, temperature, humidity, and wind speed, through the API key. In cases where the user-entered location is invalid, an alert message is displayed to indicate an error in retrieving the weather data. Furthermore, the background image of the page dynamically changes based on the prevailing weather conditions, adapting to various scenarios such as rainy, hazy, snowy, cloudy, misty, and clear sky conditions, during both day and night.
